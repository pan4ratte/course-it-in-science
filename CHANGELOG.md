# Changelog

## 1.1.0

### Обновление стилей цитирования

* Стили цитирования сихронизированы с [библиотекой](https://github.com/pan4ratte/zotero-gost-styles). Добавлено пять новых стилей:
    
    * Два дополнения к существующему с правилами сортировки библиографии по хронологии и по порядку упоминания в тексте.
    * Новый гибридный стиль формата ГОСТ 2018 с двумя альтернативными способами сортировки библиографии.
    
Подробнее см.: https://github.com/pan4ratte/zotero-gost-styles/releases/tag/3.0.0 


## 1.0.0

### Начальный релиз

* **Папка Pandoc в папке Obsidian.** Представляет собой папку пользовательских данных с минимальным набором файлов для автоматизации написания научных работ в Obsidian:

    * Папка `filters` содержит lua-фильтры: `pagebreak.lua` для обработки ручных разрывов страниц при экспорте, а также `zotero.lua` для конвертации pandoc-цитирований в live-цитирования Zotero при экспорте.
    * Файл стиля цитирования `gost2018.csl` предназначен для отображения цитирований в тексте внутри Obsidian (не влияет на экспорт).
    * Файлы `reference-gost-12.docx` и `reference-gost-14.docx` представляют собой шаблоны, которые pandoc использует для оформления документов при экспорте. Оформлены с базовым соблюдением требований для оформления работ по ГОСТ.

* **Папка Zotero.** На данный момент содержит только актуальную версию разработанного мной стиля цитирования по ГОСТ 2008 с поправкой на постраничное цитирование.
